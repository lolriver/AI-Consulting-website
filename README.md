# AI Consulting Agency Landing Page

A premium, high-performance landing page for a Strategic AI Consulting business, built with Next.js 14 and Tailwind CSS.

## üöÄ Features

-   **Premium UI/UX**: Dark-themed, glassmorphism design with a "Cyan/Blue/Indigo" palette.
-   **3D Hero Experience**: Interactive 3D background using Spline (Lazy loaded for performance).
-   **Interactive Chatbot**: Custom floating AI Assistant widget with mock conversation logic and typing indicators.
-   **Functional Contact Form**: Validated form (Zod + React Hook Form) with instant user feedback.
-   **Visual Storytelling**:
    -   **AI Console**: Live terminal animation showcasing "code execution".
    -   **Data Graphs**: Animated analytics visualizations.
    -   **Fade-Up Text**: Smooth, performance-optimized text reveals.
-   **Seamless Navigation**: Functional CTA buttons with smooth scrolling to key sections.
-   **Mobile Responsive**: Fully optimized layouts for all device sizes.

## üì¶ Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/lolriver/AI-Consulting-website.git
    ```

2.  Install dependencies:
    ```bash
    npm install
    # Fix any legacy peer deps if needed
    npm install --legacy-peer-deps
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser.
